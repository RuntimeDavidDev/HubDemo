# 招聘岗位抓取工具 — 使用手册
注意：boss反扒虫禁止使用
## 一、简介

本工具自动在 Boss 直聘、智联招聘、猎聘按关键字检索岗位，并生成本地 HTML 日报。  
支持浏览器会话持久化（一次登录，后续免登），带有交互式配置和超时保护。

## 二、环境要求

| 依赖 | 版本 |
|------|------|
| Python | ≥ 3.8 |
| playwright | ≥ 1.40.0 |
| Jinja2 | ≥ 3.1.0 |
| Chromium | playwright 自动下载 |

## 三、安装（仅首次）

打开 **PowerShell**（按 `Win + R`，输入 `powershell`，回车），逐条执行：

```powershell
# 1. 进入项目目录
cd C:\Users\David\.kun\default_workspace\job-scraper

# 2. 安装 Python 依赖
pip install -r requirements.txt

# 3. 安装 Chromium 浏览器（约 150MB）
playwright install chromium
```

> **常见问题：**
> - `'pip' 不是内部命令` → 换成 `python -m pip install -r requirements.txt`
> - `'python' 不是内部命令` → 到 [python.org](https://python.org) 装 Python 3.8+，勾选 **Add Python to PATH**，然后重新打开 PowerShell
> - `'playwright' 不是内部命令` → 换成 `python -m playwright install chromium`

## 四、日常使用

### 4.1 基本运行

```powershell
cd C:\Users\David\.kun\default_workspace\job-scraper
python main.py
```

### 4.2 运行流程概述

```
启动 → 显示配置 → 检查会话 → 确认开始 → 逐平台抓取 → 生成日报
```

每一步终端都有清晰的中文提示和进度显示。

### 4.3 命令行参数

| 参数 | 作用 | 示例 |
|------|------|------|
| （无） | 读取已有配置，直接抓取 | `python main.py` |
| `--config` / `-c` | 重新进入交互配置 | `python main.py --config` |
| `--timeout N` / `-t N` | 覆盖超时（分钟） | `python main.py --timeout 10` |
| `--headless` | 不显示浏览器窗口 | `python main.py --headless` |

## 五、交互式配置说明

首次运行（或使用 `--config`）会自动进入问答模式。每个问题都可以直接按 **Enter** 用默认值。

| 问题 | 默认值 | 示例输入 |
|------|--------|---------|
| 你想从哪些招聘网站抓取岗位？ | Boss直聘, 智联招聘, 猎聘 | 直接回车 |
| 你主要关注哪类岗位？ | (无) | `CMDB,运维开发` |
| 日报希望以什么方式发送给你？ | 本地HTML文件 | 直接回车 |
| 日报输出到哪个目录？ | 当前 output/ 目录 | 直接回车 |
| 全局超时时间（分钟）？ | 5 | `10` |
| 是否使用无头模式？(y/n) | 否 | `n` |

配置会保存到 `config.json`，下次运行不需要重新输入。

## 六、首次登录（仅第一次需要）

由于没有历史会话，脚本会依次弹出 3 个浏览器窗口：

```
[Boss直聘] 请在打开的浏览器窗口中完成登录（扫码/验证码）。
[Boss直聘] 登录完成后，回到此终端按 Enter 继续...
```

**操作步骤：**
1. 在弹出的浏览器窗口完成扫码登录 / 滑块验证
2. 确认已登录后，回到终端按 **Enter**
3. 对智联招聘、猎聘重复上述操作

登录成功后，会话自动保存到 `sessions/` 目录。  
此后运行 `python main.py` 直接进入抓取，无需再次登录。

> **会话过期：** 如果某个站点 Cookie 过期，脚本会自动检测到未登录，再次弹出浏览器要求手动登录。

## 七、抓取进度解读

运行时会看到如下日志：

```
共 3 个抓取任务 (3 平台 × 1 关键词)

[进度 1/3] [Boss直聘] 搜索 "CMDB" (剩余 2 个任务)
[Boss直聘] 启动浏览器...
[Boss直聘] 检查登录状态...
[Boss直聘] 已登录 ✓
[Boss直聘] 搜索关键词: "CMDB" ...
[Boss直聘] 解析到 15 条岗位
[Boss直聘] ✅ 获取到 15 条岗位 (12.3s)
```

| 日志片段 | 含义 |
|---------|------|
| `[进度 1/3]` | 当前是第 1 个任务，共 3 个 |
| `搜索 "CMDB"` | 当前搜索的关键词 |
| `(剩余 2 个任务)` | 完成任务后还剩 2 个 |
| `检查登录状态...` | 正在验证是否已登录 |
| `已登录 ✓` | 已有有效会话，无需手动登录 |
| `解析到 15 条岗位` | 从这个页面提取到 15 条招聘信息 |
| `✅ 获取到 15 条岗位 (12.3s)` | 任务成功，耗时 12.3 秒 |

## 八、超时保护

全局超时默认 **5 分钟**，可在配置或 `--timeout` 中修改。

当总运行时间达到限制时：

```
⏰ 已达到全局超时限制（5 分钟），终止剩余任务。
```

- 正在进行的任务会被中断
- 已完成平台的数据**不会丢失**
- 日报仍会生成，顶部有 ⚠️ 超时提示

## 九、查看日报

### 9.1 自动打开

脚本结束后会自动用默认浏览器打开日报。

### 9.2 手动打开

日报保存在 `output/` 目录，文件名格式为 `job_report_20260616_143000.html`。  
直接双击即可用浏览器查看。

### 9.3 日报内容

日报包含：
- **顶部横幅**：日期、关键词、总耗时
- **超时/异常警告**（如有）：醒目提示哪些平台未完成
- **统计卡片**：总岗位数、平台数、成功数、耗时
- **分平台表格**：岗位名称、公司、薪资、地点、经验/学历、标签

## 十、常见问题

### Q: 抓取不到岗位怎么办？

招聘网站的页面结构会变化。如果持续抓取不到：
1. 确保浏览器是**非无头模式**（能看到页面正常加载）
2. 检查搜索 URL 是否正常打开
3. 联系维护者调整选择器

### Q: 如何只抓取一个平台？

重新配置：`python main.py --config`，在"你想从哪些招聘网站抓取岗位？"时只输入需要的平台名，例如 `Boss直聘`。

### Q: 如何添加新的招聘网站？

参见《脚本程序结构手册》的「扩展指南」章节。

### Q: 会话文件在哪里？如何清除？

会话文件在 `sessions/` 目录下，按平台分文件夹：
- `sessions/boss/state.json`
- `sessions/zhilian/state.json`
- `sessions/liepin/state.json`

删除对应文件即可清除会话，下次运行会要求重新登录。
